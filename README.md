# NTE-AutoFish-Neverness To Everness
This is a script I made to be able to AutoFish on Neverness to Everness.

It runs on AutoHotKey.

You have to toggle it with F7 once you are on the "Press F" to fish.

The script casts the line, starts the minigame, fishes it for you and restarts eveything, you only have to care about running out of bait.

Keep in mind, if the UI is not set to default and/or you have changed resolution or how your screen represent colors, the script might not work. 

I don't intend to fix or improve it anymore, if you feel like doing so, feel free to do it!
